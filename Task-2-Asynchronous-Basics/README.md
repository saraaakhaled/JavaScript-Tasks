# Task 2 - Asynchronous Basics

## Objective

Practice asynchronous JavaScript using setTimeout().

## Exercises

1. Print "Hello" immediately and "World" after 2 seconds.
2. Print numbers from 1 to 5 with delays.
3. Display "Loading..." then "Done".
4. Simulate a delayed message.

## How to Run

```bash
node task1.js
node task2.js
node task3.js
node task4.js
```
