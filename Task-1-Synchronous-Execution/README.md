# Task 1 - Synchronous Execution

## Description

This task demonstrates synchronous execution in JavaScript.

### Exercises

1. Print "Start", "Middle", and "End".
2. Call one function from another.
3. Perform calculations sequentially.
4. Build a function depending on another function's result.

## Technologies

- JavaScript (ES6)
- Node.js

## How to Run

```bash
node task1.js
node task2.js
node task3.js
node task4.js
```
